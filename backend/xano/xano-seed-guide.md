# Xano Seed Data Guide — PharmaConnect v2.0

Xano supports CSV/JSON import. Use these steps to load demo data.

## Quick Seed via Xano Console

1. Go to Database > [table] > Import
2. Upload the corresponding CSV file
3. Map fields automatically
4. Confirm import

## Notes
- UUIDs: generate random UUIDs; keep them consistent across related tables.
- Dates: use ISO 8601 format (2026-07-09T10:00:00Z).
- Passwords: leave blank for OTP-only users.
- Relations: ensure referenced IDs exist before importing dependent rows.
